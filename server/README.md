# FancyTodo

**Add To Do**
----
  Save new To Do to database and return new To Do as object

* **URL**

  /todos

* **Method:**
  
  `POST`
  
* **Data Params**

    title,
    description,
    status,
    due_date

* **Success Response:**  

  * **Code:** 201 <br />
    **Content:** 
{
    "id": 6,
    "title": "Live Code3",
    "description": "Third live code",
    "status": "On Progress",
    "due_date": "2020-10-30T00:00:00.000Z",
    "updatedAt": "2020-10-26T10:44:39.247Z",
    "createdAt": "2020-10-26T10:44:39.247Z"
}
 
* **Error Response:**

  * **Code:** 400 UNAUTHORIZED <br />

**Show Todo By Id**
----
  Show Todo By Id

* **URL**

  /todos/:id

* **Method:**
  
  `GET`
  
*  **URL Params**

   **Required:**
 
   `id=[integer]`

* **Success Response:**

  * **Code:** 200 <br />
    **Content:** 
    {
      "id": 1,
      "title": "Coding",
      "description": "Coding Fancy TO DO",
      "status": "On Progress",
      "due_date": "2020-10-27T00:00:00.000Z",
      "createdAt": "2020-10-26T07:35:20.797Z",
      "updatedAt": "2020-10-26T07:35:20.797Z"
    }
 
* **Error Response:**

  * **Code:** 404 NOT_FOUND <br />
    **Content:** `{ error : "Data not found" }`
